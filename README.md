# KeyBinds

Tab: Go back to console
Esc: Close
WASD: Move
Mouse: Look around

# Console commands
'start': Starts the game
'exit': Closes the game
'shader <int>': Changes between shaders (0=None,1=Default,2=Cinematic)
'rd <int>': Changes the render distance
'light <int>': Toggles the lights (0=Off,1=On)
